# CARPACK GOUVERNEMENT

Ce pack contient 4 véhicules:
* 1 Limousine "__OneBeast__"
* 1 SUV "__Escalade__"
* 1 Jet "__Luxor3__"
* 1 Hélicoptère "__Whitehawk__"

## Description

__Escalade__:

SUV de Luxe, basé sur la Cadillac Escalade 2015.

Equipé de gyrophares + blindage + vitres pare-balles + intérieur.

__OneBeast__:

Limousine Cadillac modifié, basé sur la Cadillac One surnommé "OneBeast".

Equipé de gyrophares + blindage + vitres pare-balles + drapeaux + intérieur.

__Luxor3__:

Jet présidentiel, basé sur le Luxor2 IG.

__WhiteHawk__:

Hélicoptère basé sur le Sikorsky VH-60 White Hawk.

Equipé d'un intérieur + helicam + tableau de bord complet.

Dispose d'une entrée de configuration pour la queue de l'appareil ("trailerhawk").
